## Commands

- Print the present working directory

  ```sh
  pwd
  ````

- Generate SSH key

  ```sh
  ssh-keygen -t ed25519 -b 4096 -C "" -P "" -f ssh-identity
  ```

- Initialize terraform

  ```sh
  terraform init
  ```

- Apply configuration

  ```sh
  terraform apply
  ```

- Destroy resources

  ```sh
  terraform destroy
  ```
